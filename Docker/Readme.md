## How to Install Docker in ec2 Intance
    - docker version
    - go to websites or dockersdocs
    - sudo yum install docker -y
    - sudo systemctl start docker
    - sudo systemctl enable docker
    - docker version
    - docker ps 
    - sudo chmod 777 /var/run/dockers.sock
    - sudo usermod -aG docker ec2-user
    - newgrp docker
